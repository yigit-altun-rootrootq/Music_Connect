WHAT CAN IT DO?

🎛️

100% Blueprint Friendly

No C++ required. Drag, drop, connect nodes — works in any Blueprint Actor, GameMode, or PlayerController.

🌐

Multi-Platform Streaming

Search by name or paste a URL from YouTube, Spotify, SoundCloud, or Apple Music. It just works.

⏯️

Full Playback Control

Play, Pause, Resume, Stop, Next/Previous, Jump To Track, Loop, Shuffle — all as Blueprint nodes.

🔊

Volume & Fade

Set volume instantly (0.0–1.0) or fade in/out smoothly over time with a single node.

📡

Event Dispatchers

React automatically to On State Changed, On Track Changed, and On Error — perfect for dynamic UIs.

📦

Packaged Game Ready

One extra Project Settings entry and your shipped game keeps streaming music without issues.

// Quick Start

UP AND RUNNING IN 60 SECONDS

01

Add the Component

Open your Blueprint Actor, go to Components, click + Add, search for Music Connect Player and add it.

02

Drop the Play Node

Drag the component into the Event Graph, pull a wire and search Play Music Stream. Type a song name or paste a URL.

03

Wire Your Events

Connect On Playing to update UI, On Error for fallbacks, and On Track Changed for playlist logic.

04

Package Your Game

In Project Settings add Plugins/MusicConnect/Binaries/ThirdParty to Additional Non-Asset Directories. Done!
